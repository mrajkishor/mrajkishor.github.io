This code creates a neural network with 2 input neurons, 3 hidden neurons, and 1 output neuron, and trains it on some random data using backprop.

	import numpy as np

	class NeuralNetwork:
	    def __init__(self, layers):
	        self.layers = layers
	        self.weights = []
	        self.biases = []
	        for i in range(1, len(layers)):
	            self.weights.append(np.random.randn(layers[i], layers[i-1]))
	            self.biases.append(np.random.randn(layers[i], 1))
	    
	    def sigmoid(self, x):
	        return 1 / (1 + np.exp(-x))
	    
	    def forward(self, x):
	        a = np.copy(x)
	        self.z = []
	        self.a = [a]
	        for i in range(len(self.layers) - 1):
	            z = np.dot(self.weights[i], a) + self.biases[i]
	            self.z.append(z)
	            a = self.sigmoid(z)
	            self.a.append(a)
	        return a
	    
	    def sigmoid_prime(self, x):
	        return self.sigmoid(x) * (1 - self.sigmoid(x))
	    
	    def cost_derivative(self, output_activations, y):
	        return output_activations - y
	    
	    def backpropagation(self, y):
	        delta = self.cost_derivative(self.a[-1], y) * self.sigmoid_prime(self.z[-1])
	        nabla_w = [np.dot(delta, self.a[-2].T)]
	        nabla_b = [delta]
	        for i in range(2, len(self.layers)):
	            delta = np.dot(self.weights[-i+1].T, delta) * self.sigmoid_prime(self.z[-i])
	            nabla_w.append(np.dot(delta, self.a[-i-1].T))
	            nabla_b.append(delta)
	        nabla_w.reverse()
	        nabla_b.reverse()
	        return nabla_w, nabla_b
	    
	    def train(self, X, Y, epochs, learning_rate):
	        for i in range(epochs):
	            cost = 0
	            for j in range(len(X)):
	                x = X[j].reshape(len(X[j]), 1)
	                y = Y[j].reshape(len(Y[j]), 1)
	                output = self.forward(x)
	                cost += np.linalg.norm(output - y)
	                nabla_w, nabla_b = self.backpropagation(y)
	                for k in range(len(self.weights)):
	                    self.weights[k] -= learning_rate * nabla_w[k]
	                    self.biases[k] -= learning_rate * nabla_b[k]
	            print("Epoch {}: cost = {}".format(i+1, cost))
	    
	    def predict(self, x):
	        output = self.forward(x.reshape(len(x), 1))
	        return output


Here's an example usage of this neural network:
		
	# Create a neural network with 2 input neurons, 3 hidden neurons, and 1 output neuron
	net = NeuralNetwork([2, 3, 1])

	# Generate some random training data
	X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
	Y = np.array([[0], [1], [1], [0]])

	# Train the neural network on the data for 1000 epochs with a learning rate of 0.1
	net.train(X, Y, 1000, 0.1)

	# Predict the output for a new input
	x = np.array([0, 1])
	output = net.predict(x)
	print("Input: {}\nOutput: {}".format(x, output))

2.  Define a `NeuralNetwork` class with the following methods:
    -   `__init__`: Initializes the neural network with a given list of layer sizes, where the first element of the list is the input layer size and the last element is the output layer size. Randomly initializes the weights and biases for each layer using the normal distribution.
    -   `sigmoid`: Computes the sigmoid activation function for a given input.
    -   `forward`: Performs forward propagation through the neural network for a given input and returns the output.
    -   `sigmoid_prime`: Computes the derivative of the sigmoid function for a given input.
    -   `cost_derivative`: Computes the derivative of the cost function (which in this case is just the squared error) with respect to the output activations.
    -   `backpropagation`: Performs backpropagation through the neural network for a given input and target output, and returns the gradients of the weights and biases for each layer.
    -   `train`: Trains the neural network on a given set of inputs and target outputs for a given number of epochs using a given learning rate. Uses backpropagation to update the weights and biases for each layer.
    -   `predict`: Predicts the output for a given input by performing forward propagation through the neural network.
3.  In the `__init__` method, initialize the weights and biases for each layer using the normal distribution.
4.  In the `sigmoid` method, compute the sigmoid activation function for a given input using the NumPy `exp` function.
5.  In the `forward` method, perform forward propagation through the neural network by computing the weighted sum of the input activations for each layer, applying the sigmoid activation function, and storing the results.
6.  In the `sigmoid_prime` method, compute the derivative of the sigmoid function for a given input using the sigmoid function itself.
7.  In the `cost_derivative` method, compute the derivative of the squared error cost function with respect to the output activations using the NumPy `-` operator.
8.  In the `backpropagation` method, perform backpropagation through the neural network by computing the error at each layer using the chain rule, and using these errors to compute the gradients of the weights and biases for each layer.
9.  In the `train` method, iterate through the training data for a given number of epochs, and for each input/target pair, perform forward propagation to compute the output, backpropagation to compute the gradients, and update the weights and biases for each layer using the gradients and a given learning rate.
10.  In the `predict` method, perform forward propagation through the neural network for a given input to compute the output, and return the result.
