**Default props** are predefined values assigned to a componentâ€™s props in case the **parent component does not supply them**. They ensure that the component behaves predictably even when certain props are omitted.

Default props help:

* Avoid `undefined` or `null` issues
* Set fallback values
* Improve code readability and reliability

---

#### ðŸ”¹ Basic Example (JavaScript)

```jsx
const Welcome = ({ name }) => {
  return <h1>Hello, {name}</h1>;
};

Welcome.defaultProps = {
  name: 'Guest'
};
```

Usage:

```jsx
<Welcome /> // Output: Hello, Guest
<Welcome name="Alice" /> // Output: Hello, Alice
```
