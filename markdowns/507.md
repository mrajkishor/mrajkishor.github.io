## ðŸ›  2. Create a GraphQL API Route

### `pages/api/graphql.js`

```js
import { createSchema, createYoga } from 'graphql-yoga';

const typeDefs = /* GraphQL */ `
  type Query {
    hello: String
    getUser(id: ID!): User
  }

  type Mutation {
    createUser(name: String!, email: String!): User
  }

  type User {
    id: ID!
    name: String!
    email: String!
  }
`;

let users = [];

const resolvers = {
  Query: {
    hello: () => 'Hello from GraphQL API!',
    getUser: (_, { id }) => users.find((u) => u.id === id),
  },
  Mutation: {
    createUser: (_, { name, email }) => {
      const user = { id: `${Date.now()}`, name, email };
      users.push(user);
      return user;
    },
  },
};

const yoga = createYoga({
  schema: createSchema({ typeDefs, resolvers }),
  graphqlEndpoint: '/api/graphql',
});

export default yoga;
```

> You now have a live GraphQL server at `http://localhost:3000/api/graphql`.

## ðŸš€ 3. Querying From the Frontend

You can use plain `fetch()` or a GraphQL client like Apollo or urql.

### Example using `fetch`:

```js
async function fetchUser(id) {
  const res = await fetch('/api/graphql', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      query: `
        query {
          getUser(id: "${id}") {
            name
            email
          }
        }
      `,
    }),
  });
  const { data } = await res.json();
  return data.getUser;
}
```
