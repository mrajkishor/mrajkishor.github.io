### ðŸ“Œ **2. Find Intersection of Two Linked Lists**

* Use two pointers `a` and `b` starting at each list's head.
* Traverse each list, and then switch to the other list when reaching the end.
* When they meet, thatâ€™s the **intersection node**.

```java
ListNode a = headA;
ListNode b = headB;

while (a != b) {
    a = (a != null) ? a.next : headB;
    b = (b != null) ? b.next : headA;
}
return a; // intersection node or null
```

    ListNode next;

    ListNode(int val) {
        this.val = val;
    }

    public static ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode a = headA;
        ListNode b = headB;

        while (a != b) {
            a = (a != null) ? a.next : headB;
            b = (b != null) ? b.next : headA;
        }
        return a; // Intersection node or null
    }

    public static void main(String[] args) {

        ListNode a1 = new ListNode(3);

        ListNode b1 = new ListNode(99);
