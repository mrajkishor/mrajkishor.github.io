

### **Definition of LCM**

The **Least Common Multiple (LCM)** of two numbers is:

üëâ The **smallest positive integer** that is divisible by both numbers.

So instead of looking at **divisors** (like in GCD), we look at **multiples**.

---

### **Ways to Calculate LCM**

#### 1. **Listing Multiples (basic way)**

Example: `n1 = 4, n2 = 6`

* Multiples of 4 ‚Üí 4, 8, 12, 16, 20, 24 ‚Ä¶
* Multiples of 6 ‚Üí 6, 12, 18, 24 ‚Ä¶
* First common multiple = **12**
  ‚úÖ LCM = 12

---

#### 2. **Relation with GCD (fast way)**

There‚Äôs a neat formula:

$$
LCM(a, b) \times GCD(a, b) = a \times b
$$

or

$$
LCM(a, b) = \frac{a \times b}{GCD(a, b)}
$$

Example: `a = 12, b = 18`

* GCD(12, 18) = 6
* LCM = (12 √ó 18) √∑ 6 = 216 √∑ 6 = **36**

---

#### 3. **Prime Factorization**

Break both numbers into primes, then take the **highest power** of each prime.

Example: `12 and 18`

* 12 = 2¬≤ √ó 3¬π
* 18 = 2¬π √ó 3¬≤
* LCM = 2¬≤ √ó 3¬≤ = 36

---

### ‚úÖ Quick Comparison

* **GCD ‚Üí greatest divisor common to both.**
* **LCM ‚Üí least multiple common to both.**

---

Let‚Äôs write a clean **JavaScript function** for LCM using the GCD formula.


### **Code**

```js
// Function to find GCD using Euclidean Algorithm
function gcd(a, b) {
    while (b !== 0) {
        [a, b] = [b, a % b];
    }
    return a;
}

// Function to find LCM using GCD
function lcm(a, b) {
    return (a * b) / gcd(a, b);
}

// Example
console.log(lcm(12, 18)); // 36
console.log(lcm(4, 6));   // 12
console.log(lcm(7, 5));   // 35
```

---

### **How it works**

* First, `gcd(a, b)` is found using the efficient Euclidean algorithm.
* Then apply formula:

  $$
  LCM(a, b) = \frac{a \times b}{GCD(a, b)}
  $$
* Works for all positive integers.

---

‚ö° This approach runs in **O(log(min(a, b)))**, much faster than listing multiples.

---

IF you don‚Äôt want the **LCM via GCD formula** ‚Äî let‚Äôs write it directly.

The most straightforward way is:
üëâ Start from the **larger number** of the two and keep checking its multiples until you find one that‚Äôs divisible by the other.

---

### **Code: Direct LCM**

```js
function lcm(a, b) {
    // Start with the greater of the two
    let multiple = Math.max(a, b);

    // Keep incrementing until it's divisible by both
    while (true) {
        if (multiple % a === 0 && multiple % b === 0) {
            return multiple;
        }
        multiple++;
    }
}

// Examples
console.log(lcm(12, 18)); // 36
console.log(lcm(4, 6));   // 12
console.log(lcm(7, 5));   // 35
```

---

### **How it works**

* Start from the bigger number (`max(a, b)`), because the LCM can‚Äôt be smaller than that.
* Check if it‚Äôs divisible by both numbers.
* If yes ‚Üí return it.
* If not ‚Üí increase by 1 and try again.

---

### ‚ö†Ô∏è Note

* This works fine for small/medium numbers.
* But for very large inputs, it becomes **slow** since it may loop a lot.
* That‚Äôs why the **GCD-based formula** is preferred in practice.

```js
function gcd(a, b) {
    let result = 1;
    for (let i = 1; i <= Math.min(a, b); i++) {
        if (a % i === 0 && b % i === 0) {
            result = i;
        }
    }
    return result;
}
```

Let‚Äôs carefully explain the HCF and LCM process using the working steps from the image.


## **Step 1: Common Division Method (for HCF)**

You started with numbers **60 and 48**:

* Divide both by 2 ‚Üí (30, 24)
* Divide again by 2 ‚Üí (15, 12)
* Divide again by 3 ‚Üí (5, 4)

üëâ The divisors used here (2, 2, 3) are the **common prime factors**.
Multiply them:

$$
HCF = 2 √ó 2 √ó 3 = 12
$$

So, **HCF(60, 48) = 12**.

---

## **Step 2: Prime Factorization**

Also in the image:

* 60 = $2^2 √ó 3 √ó 5$
* 48 = $2^4 √ó 3$

Now, for **HCF**:

* Take only the **common prime factors with the smallest power**.
* Common factors are $2^2$ and $3$.
* Multiply: $2^2 √ó 3 = 12$.
  ‚úÖ Matches the first method.

---

## **Step 3: LCM**

For **LCM**, you do the opposite:

* Take **all prime factors present**, but use the **highest power** of each.

From above:

* 60 = $2^2 √ó 3 √ó 5$
* 48 = $2^4 √ó 3$

Now pick highest powers:

$$
LCM = 2^4 √ó 3 √ó 5 = 240
$$

So, **LCM(60, 48) = 240**.

---

## **Step 4: Relation Between HCF and LCM**

For two numbers:

$$
HCF √ó LCM = Product \ of \ the \ numbers
$$

Check:

$$
12 √ó 240 = 2880 \quad \text{and} \quad 60 √ó 48 = 2880
$$

‚úÖ Relation is correct.

---

## **==Key Idea==**

* **HCF** ‚Üí Product of **common prime factors with smallest powers**.
* **LCM** ‚Üí Product of **all prime factors with highest powers**.
