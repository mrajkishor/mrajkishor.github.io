## ðŸ§  What is a Promise?

A **Promise** is an object representing the eventual **completion or failure** of an asynchronous operation.

### Basic Structure:

```js
const promise = new Promise((resolve, reject) => {
  // async work
  if (success) {
    resolve(result); // fulfilled
  } else {
    reject(error);   // rejected
  }
});
```

## ðŸš€ Using Promises in Express Route Handlers

### âŒ Bad: Callback Style

```js

  db.findAllUsers((err, users) => {

  });

```

### âœ… Good: Promise Style

```js

  db.findAllUsers()
    .then(users => res.json(users))
    .catch(next); // Pass error to Express error handler
});
```

## ðŸ” Chaining Promises

```js

  getUser(req.query.id)
    .then(user => getPostsByUser(user.id))
    .then(posts => res.json(posts))
    .catch(next);
});
```

Each `.then()` returns a new Promise, allowing clean sequential flows.
