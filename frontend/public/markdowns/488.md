    int n = wt.length;
    int[][] dp = new int[n + 1][W + 1];

    for (int i = 1; i <= n; i++) {
        for (int w = 1; w <= W; w++) {
            if (wt[i-1] <= w)
                dp[i][w] = Math.max(dp[i-1][w], dp[i-1][w - wt[i-1]] + val[i-1]);

                dp[i][w] = dp[i-1][w];
        }
    }
    return dp[n][W];
}

## ğŸ¯ 2. **Subset Sum Problem**

### Problem Statement:
Given an array of integers and a sum `target`, determine if a **subset adds up** to `target`.

### ğŸ“ DP State
`dp[i][s]` = `true` if a subset of the first `i` elements has sum `s`

---

### ğŸ§± Tabulation Code (Java):
```java
boolean subsetSum(int[] nums, int sum) {
    int n = nums.length;
    boolean[][] dp = new boolean[n + 1][sum + 1];

    for (int i = 0; i <= n; i++) dp[i][0] = true;

    for (int i = 1; i <= n; i++) {
        for (int s = 1; s <= sum; s++) {
            if (nums[i-1] <= s)
                dp[i][s] = dp[i-1][s] || dp[i-1][s - nums[i-1]];
            else
                dp[i][s] = dp[i-1][s];
        }
    }
    return dp[n][sum];
}
```
