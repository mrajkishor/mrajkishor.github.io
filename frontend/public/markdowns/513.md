## ğŸ’¤ What is Lazy Loading?

**Lazy loading** delays loading a component, image, or data until it's actually needed â€” e.g., when it enters the viewport or is clicked.

## ğŸ¯ Lazy Loading Components in Next.js

Use Next.js's built-in `next/dynamic` for component-level lazy loading.

```js
import dynamic from 'next/dynamic';

const HeavyComponent = dynamic(() => import('../components/HeavyComponent'), {
  loading: () => <p>Loading...</p>,
  ssr: false,
});

export default function Page() {
  return (
    <div>
      <h1>Page with Lazy Loaded Component</h1>
      <HeavyComponent />
    </div>
  );
}
```

> âœ… `ssr: false` disables server-side rendering for that component  
> âœ… `loading` shows a fallback during load
